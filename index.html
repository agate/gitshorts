Hey! This is Gitshorts.

Hate every day git pull and push stuff? Gitshorts provide a set of commands to help you make your git life easier.

How to Install

Clone it

$ git clone git://github.com/agate/gitshorts.git
$ cd gitshorts
$ ./install

Use Mac and Homebrew?

$ brew install gitshorts
