Hey! This is Gitshot.

Hate every day dealing with the git pull and push stuff? Gitshot provide a set of commands to help you make your git life easier.

How to Install

Clone it

$ git clone git://github.com/agate/gitshot.git
$ cd gitshot
$ ./install

Use Mac and Homebrew?

$ brew install gitshot
